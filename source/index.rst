Introduction
------------
Ce projet consiste en un chatbot développé pour assister aux travaux pratiques (TP) à l'ENSAM Meknès.

Structure du projet
-------------------
Voici la structure des fichiers du projet :
- actions/
- data/
- models/
- etc.

Installation
------------
Pour installer le projet, suivez ces étapes :
1. Clonez le repository
2. Installez les dépendances avec Docker
3. Démarrez le chatbot avec Docker Compose
